# 使用tensorflow2.0实现经典网络架构

Implementations of neural network models with tf (>=2.0)

## Requirements

* TensorFlow >= 2.0
* Numpy >=1.18

```shell
$ pip install tensorflow-gpu numpy
```

## Models

* Logistic Regression
* LeNet
* ResNet (ResNet50)
* DenseNet (DenseNet121)
* Generative Adversarial Network
